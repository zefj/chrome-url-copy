Chrome extension that copies the current tab's URL to your clipboard with a keyboard shortcut or a click.

Totally vibe coded for personal use.

## Install

1. Clone this repo
2. Open `chrome://extensions`
3. Enable **Developer mode** (top right)
4. Click **Load unpacked** and select the cloned folder

## Usage

- **Keyboard shortcut:** `Cmd+Shift+C` (Mac) / `Ctrl+Shift+C` (Windows/Linux)
- **Click** the extension icon in the toolbar

A checkmark badge appears briefly to confirm the copy.

## Remap the shortcut

Go to `chrome://extensions/shortcuts` and update **Copy current tab URL**.
